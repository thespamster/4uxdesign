About 4uxdesign

Hi, I'm Gavin and '4uxdesign' is my freelance full-stack web development business. My design philosophy is a simple one. As a fan of Occam's razor I believe that the simplest solution is very often the best one. Simple code is efficient code. This doesn't mean leaving features out but rather making sure that the features included are the ones that are needed.

Solutions that are coded efficiently work better, have fewer bugs and will serve their purpose for longer. This will save you money and who doesn't want that?

Here's the nerdy bit. I build solutions using combinations of HTML5, CSS & Javascript. I use libraries such as Bootstrap and jQuery, frameworks incl. Flask and Django combined with PostgreSQL databases utilising SQL Alchemy. This is by no means an exhaustive list and you may be wondering what it all means. The TLDR; is that these technologies allow me to help you by building the website you need.

I maintain my profile on Github a website that developers use to store and share code.

My profile can be found here: Github Profile

Whether you are looking for a striking landing page or a full e-commerce solution I can help you by building a site that is tailored to your needs. I build bespoke solutions at a great price.
