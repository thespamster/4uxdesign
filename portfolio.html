The Blue Box Shop

This was the fourth and final project for my diploma studies. A full-stack e-commerce application with a responsive frontend built using HTML5, CSS3, Javascript and Bootstrap5 plus jQuery3.7. The backend was built using the Django framework, custom Python3 with a PostgreSQL database. The site featured an online shop with a custom built checkout utilising Stripe for payments. Users had the option to create an account and login to view their order history. A site admin could add, edit and delete products, categories, users and orders. Final deployment was on Heroku, the static images hosted on AWS and the database on Tembo.io. The site was live when graded but has since been taken down.
