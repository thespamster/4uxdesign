About 4uxdesign.com

Hi, I'm Gavin and my design philosophy is a simple one. As a fan of Occam's razor I believe that the simplest solution is very often the best one. Simple code is efficient code. This doesn't mean leaving features out but rather making sure that the features included are the ones that are needed.

Solutions that are coded efficiently work better, have fewer bugs and will serve their purpose for longer. This will save you money and who doesn't want that?

Here's the nerdy bit. I build solutions using combinations of HTML5, CSS & Javascript. I use libraries such as Bootstrap and jQuery, frameworks incl. Flask and Django combined with databases utilising SQL Alchemy and PostgreSQL.

I maintain my profile on Github a website that developers use to store and share code.
