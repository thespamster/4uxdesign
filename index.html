About 4uxdesign

Hi, I'm Gavin and '4uxdesign' is my freelance full-stack web development business. I am based in Buckfastleigh, Devon, UK but work globally. My design philosophy is a simple one. As a fan of Occam's razor I believe that the simplest solution is very often the best one. Simple code is efficient code. This doesn't mean leaving features out but rather making sure that the features included are the ones that are needed.

Performance and longevity come from solutions that are coded efficiently This will save you money over the lifetime of the product and who doesn't want that?

Here's the nerdy bit. I build solutions using combinations of HTML5, CSS3, Javascript, Python3 and others. I use libraries such as Bootstrap5, Tailwind and jQuery3.7 slim, frameworks incl. Flask and Django combined with PostgreSQL databases. This is by no means an exhaustive list and you may be wondering what it all means.

The TLDR; is that these technologies allow me to help you by building the website you need.

I maintain my profile on Github a website that developers use to store and share code.

My profile can be found here: Github Profile

Whether you are looking for a striking landing page or a full e-commerce solution I can help you by building a site that is tailored to your needs. I build bespoke solutions at a great price. As a full-stack web developer I understand both the frontend (what you see in your browser) and the backend (the bit that stores the data, user information, product details etc).

The Blue Box Shop

This was the fourth and final project for my diploma studies. A full-stack e-commerce application with a responsive frontend built using HTML5, CSS3, Javascript and Bootstrap5 plus jQuery3.7. The backend was built using the Django framework, custom Python3 with a PostgreSQL database. The site featured an online shop with a custom built checkout utilising Stripe for payments. Users had the option to create an account and login to view their order history. A site admin could add, edit and delete products, categories, users and orders. Final deployment was on Heroku, the static images hosted on AWS and the database on Tembo.io. The site was live when graded but has since been taken down.
